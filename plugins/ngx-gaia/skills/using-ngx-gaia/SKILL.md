# ngx-gaia

Requires `@vsn-ux/ngx-gaia`. If doc links fail, the package is not installed in this project.

## Patterns

- Elements: `<ga-alert>`, `<ga-button>`, `<ga-select>`
- Directives: `[gaButton]`, `[gaInput]`, `[gaTooltip]`
- Modules: `Ga{Component}Module` (e.g., `GaButtonModule`)
